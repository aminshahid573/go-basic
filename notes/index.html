1. Hello World

Your First Go Program

package main

import "fmt"

func main(){
    fmt.Println("Hello, World!")
}
  • package main: Declares the package name. Every Go program starts with a package declaration. main is special: it tells Go this is an executable program, not a shared library.
  • import "fmt": Imports the fmt package, which contains functions for formatted I/O (input/output), like printing to the console.
  • func main(): The entry point of every Go program. The main function is where execution starts.
  • fmt.Println("Hello, World!"): Calls the Println function from the fmt package to print text to the console, followed by a newline.
  • Note: Go uses curly braces {} for code blocks, and all statements end with a newline (not a semicolon, except in rare cases).
  • Extra: Try changing the text inside Println to print your name or any message!
  • Did you know? Go files must be saved with a .go extension, and the main function is required for standalone programs.
Quiz: What happens if you remove the main function from this program?
Show Answer
If you remove main, the program will compile but not run, because Go looks for func main() as the entry point.

2. Simple Values

Working with Different Types

package main

import "fmt"

func main() {
    //integer
    fmt.Println(1)
    fmt.Println(1 + 1)

    //string
    fmt.Println("Hello, World!")

    //bool
    fmt.Println(true)
    fmt.Println(false)

    //floats
    fmt.Println(10.5)
    fmt.Println(7.0 / 2.3)
}
  • Integers: fmt.Println(1) prints an integer. Go supports various integer types: int, int8, int16, int32, int64.
  • Arithmetic: fmt.Println(1 + 1) shows addition. Go supports all basic arithmetic operations.
  • Strings: fmt.Println("Hello, World!") prints a string. Strings in Go are UTF-8 encoded.
  • Booleans: fmt.Println(true) and fmt.Println(false) print boolean values.
  • Floats: fmt.Println(10.5) and fmt.Println(7.0 / 2.3) print floating-point numbers. Go has float32 and float64.
  • Extra: Try changing the numbers or types to see what happens. What if you try fmt.Println("3" + "4")?
  • Did you know? Go is statically typed, so you can't mix types in operations (e.g., 1 + "1" is not allowed).
Quiz: What happens if you try to add a string and an integer in Go?
Show Answer
Go will give a compile-time error: you can't add different types directly.

3. Variables

Declaring and Using Variables

package main

import "fmt"

func main() {
    // String
    var name string = "Shahid"
    var name2 = "Amin" // this is also valid
    fmt.Println(name, name2)

    //boolean
    var isAdult bool = true
    fmt.Println(isAdult)

    //integer
    var age int = 30
    fmt.Println(age)

    //float
    var height float64 = 5.9
    fmt.Println(height)

    //variable with no value
    var country string
    fmt.Println(country)

    //Shorthand declaration
    //best for declaring variable and assign value immediately
    city := "Biratnagar"
    fmt.Println(city)
}
  • var name string = "Shahid": Declares a string variable with an explicit type and value.
  • var name2 = "Amin": Type inference lets Go figure out the type from the value.
  • var isAdult bool = true: Boolean variable.
  • var age int = 30: Integer variable.
  • var height float64 = 5.9: Floating-point variable.
  • var country string: Declared but not initialized, so its value is the zero value for strings: "" (empty string).
  • city := "Biratnagar": Short variable declaration, only allowed inside functions.
  • Extra: Try declaring variables of other types, or without initializing them. What value do they get?
  • Did you know? All variables in Go have a zero value if not initialized (e.g., 0 for int, false for bool, "" for string).
Quiz: What is the zero value of a boolean variable in Go?
Show Answer
The zero value for a boolean is false.

4. Constants

Declaring Constants

package main

import "fmt"

//shorthand syntax wont work in global
const age = 30

func main() {
    const name string = "golang"

    // name = "javascript" // This will cause an error because constants cannot be reassigned

    fmt.Println("Name:", name)
    fmt.Println("Age:", age)

    // Constants can be declared in a group
    const (
        port = 5000
        host = "localhost"
    )

    fmt.Println(port, host)
}
  • const age = 30: Declares a constant at the package level. Constants cannot be changed after declaration.
  • const name string = "golang": Declares a string constant inside main.
  • name = "javascript": This line is commented out because constants cannot be reassigned.
  • const ( ... ): Declares multiple constants in a group.
  • Extra: Try changing a constant's value and see what error you get.
  • Did you know? Constants can only be numbers, strings, or booleans (not slices, maps, etc.).
Quiz: What happens if you try to assign a new value to a constant in Go?
Show Answer
Go will give a compile-time error: constants cannot be reassigned.

5. For Loops

Looping in Go

package main

import "fmt"

//for -> only contruct in go looping
func main() {
    //while loop
    i := 1
    for i <= 3 {
        fmt.Println(i)
        i++
    }

    //infinite loop
    // for {
    //     fmt.Println("Infinite Loop")
    // }

    //classic for loop
    // also can use continue and break
    for j := 0; j < 3; j++ {
        fmt.Println(j)
    }

    // range loop
    for i := range 3 {
        fmt.Println(i)
    }
}
  • for i <= 3: Go's for loop can act like a while loop.
  • for { ... }: Infinite loop (commented out here).
  • for j := 0; j < 3; j++: Classic C-style for loop.
  • continue and break can be used in loops (not shown here, but try adding them!).
  • for i := range 3: Loops from 0 to 2 (Go 1.22+ feature!).
  • Extra: Try uncommenting the infinite loop (but be careful!).
  • Did you know? Go only has one loop keyword: for. There is no while or do-while.
Quiz: What happens if you forget to increment i in the first loop?
Show Answer
The loop will run forever (infinite loop) because i never changes.

6. If/Else

Conditional Logic

package main

import "fmt"

func main() {
    // age := 16

    // if age >= 18 {
    //     fmt.Println("You are an adult.")
    // } else {
    //     fmt.Println("You are not an adult.")
    // }

    percentage := 40

    if percentage > 65 {
        fmt.Println("B+")
    } else if percentage > 80 {
        fmt.Println("A")
    } else if percentage <= 65 {
        fmt.Println("Fail")
    } else {
        fmt.Println("A+")
    }

    var role = "admin"
    var hasPermission = true

    if role == "admin" || hasPermission {
        fmt.Println("User has admin permission")
    }

    //we can declare variable in if statement
    if age := 15; age >= 18 {
        fmt.Println("You are an adult.")
    } else {
        fmt.Println("You are not an adult.")
    }

    //go doesnot have ternary operator, you will have to use normal if else
}
  • if percentage > 65: Standard if/else if/else chain.
  • role == "admin" || hasPermission: Logical OR, checks if either condition is true.
  • if age := 15; age >= 18: You can declare a variable just for the if statement.
  • No ternary operator: Go does not have condition ? a : b; use if/else instead.
  • Extra: Try changing the values and see what gets printed.
  • Did you know? All conditions in Go must be boolean (no implicit conversion from int, etc.).
Quiz: Can you use a ternary operator in Go?
Show Answer
No, Go does not have a ternary operator. Use if/else instead.

7. Switch

Switch Statements

package main

import (
    "fmt"
    "time"
)

func main() {
    //simple switch
    i := 5

    //break keyword is not necessary
    switch i {
    case 1:
        fmt.Println("one")
    case 2:
        fmt.Println("two")
    case 3:
        fmt.Println("three")
    default: //optional
        fmt.Println("other")
    }

    //multiple condition switch
    switch time.Now().Weekday() {
    case time.Saturday, time.Sunday:
        fmt.Println("It's the weekend!")
    default:
        fmt.Println("It's a weekday.")
    }

    //type switch
    // interface{} means any type
    whoAmI := func(i interface{}) {
        switch t := i.(type) { //we can also write `switch i.(type)` if we dont need to store the type
        case int:
            fmt.Println("its an integer")
        case string:
            fmt.Println("its a string")
        case bool:
            fmt.Println("its a boolean")
        default:
            fmt.Printf("other type : %T", t)
        }
    }

    whoAmI(42.21)
}
  • switch i: Switches on the value of i. No break needed; Go breaks automatically.
  • default: Optional, runs if no case matches.
  • switch time.Now().Weekday(): Switch on the current day of the week. Multiple values per case allowed.
  • Type switch: switch t := i.(type) checks the type of i (useful for interfaces).
  • Extra: Try adding more cases or switching on different types.
  • Did you know? Go's switch is more flexible than in C/Java: you can switch on any type, not just integers.
Quiz: What happens if no case matches and there is no default?
Show Answer
Nothing happens; the switch statement is skipped.

8. Arrays

Arrays in Go

package main

import "fmt"

func main() {
    //array is fixed size collection of elements of the same type
    // memory optimization
    // very fast to access
    var nums [4]int

    fmt.Println(len(nums)) // Length of the array

    nums[0] = 1

    fmt.Println(nums[0])
    fmt.Println(nums)

    var name [3]string
    name[0] = "John"
    name[1] = "Doe"
    name[2] = "Smith"

    fmt.Println(name)

    //another way to declare and initialize an array
    colors := [3]string{"Red", "Green", "Blue"}
    fmt.Println(colors)

    // Multidimensional array
    multiDimension := [2][2]int{{3, 4}, {5, 6}}
    fmt.Println(multiDimension)
}
  • var nums [4]int: Declares an array of 4 integers. Arrays are fixed size.
  • len(nums): Returns the length of the array.
  • nums[0] = 1: Assigns a value to the first element.
  • var name [3]string: Array of 3 strings.
  • colors := [3]string{"Red", "Green", "Blue"}: Declare and initialize an array in one line.
  • multiDimension := [2][2]int{{3, 4}, {5, 6}}: Multidimensional array.
  • Extra: Try changing the size or type of the array.
  • Did you know? Arrays are value types in Go: assigning one array to another copies all elements.
Quiz: What happens if you try to access an index outside the array's bounds?
Show Answer
Go will panic at runtime with an "index out of range" error.

9. Slices

What is a Slice?

var nums []int // nil slice (no underlying array yet)
Slice is a dynamic, flexible view into an array. Unlike arrays, slices can grow and shrink.
Uninitialized slices are nil.

Creating Slices

nums := make([]int, 2) // [0 0]
Use make to create a slice with a given length (and optional capacity).
Best practice: Use make([]T, 0, cap) if you know the expected size.

Appending Elements

nums = append(nums, 42)
append adds elements to a slice. If the slice is at capacity, Go allocates a new underlying array.

Slice Length & Capacity

len(nums) // number of elements
cap(nums) // capacity before reallocation
len returns the number of elements. cap returns the capacity (max before reallocation).

Copying Slices

copy(dst, src)
copy copies elements from src to dst. Both must be slices of the same type.

Slicing Syntax

nums[1:3] // elements 1 and 2 (excludes 3)
Slicing returns a new slice referencing the same array.
nums[:n] is from start to n-1, nums[m:] is from m to end.

Comparing Slices

slices.Equal(a, b) // Go 1.18+
Use slices.Equal to compare two slices for equality (requires import "slices" in Go 1.18+).

Multidimensional Slices

matrix := [][]int{{1,2,3},{4,5,6}}
Slices of slices can be used for 2D data structures.

What else should you know?

  • Zero value: A nil slice behaves like an empty slice in most cases.
  • Appending multiple elements: append(nums, 1, 2, 3)
  • Removing elements: Use nums = append(nums[:i], nums[i+1:]...) to remove the i-th element.
  • Best practice: Avoid sharing slices between goroutines without synchronization.
