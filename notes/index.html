1. Hello World

What is a Go Program?

Every Go program consists of:

  • package main - Declares the package name
  • import statements - Include external packages
  • func main() - Entry point of the program

Your First Go Program

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

2. Simple Values

Basic Data Types in Go

Go provides several basic data types:

  • Integers: int, int8, int16, int32, int64
  • Floating-point: float32, float64
  • Strings: UTF-8 encoded text
  • Booleans: true or false

Working with Different Types

package main

import "fmt"

func main() {
    // Numbers
    fmt.Println(42)           // Integer
    fmt.Println(3.14)         // Float
    
    // Strings
    fmt.Println("Go is fun!") // String
    
    // Booleans
    fmt.Println(true)         // Boolean
    
    // Basic arithmetic
    fmt.Println(10 + 5)       // Addition
    fmt.Println(10 * 5)       // Multiplication
}
